HTML

  • html lang="en":lets the browser know that the document is written in english
  • head: is not visable to user behind the scenes functionallity
  • meta: meta tag is used for metadata which helps with SEO (the ranking on which a website shows up on a browser)
  • title: sets the title of webpage

CSS

  • CSS can set the position of elements on the page.
  • CSS can change the color
  • CSS can change the font
  • CSS can change the text type
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git can be used to link your remote repositor to your local repo
  • Git can be used to generate SSH keys
  • Git can be used to create foloders and navigate threw your computers directory
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git switch -c branchName: creates a new branch name and switches to it.
  • git add .: updates changed files to be ready for commit.
